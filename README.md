# ECS Indy — Commercial Grease Exhaust Specialists

Professional website for **ECS Indy** (ECSINDY.COM), a commercial kitchen exhaust cleaning company based in Greenwood, Indiana.

## 🚀 Deploy to Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start)

1. Push this repo to GitHub
2. Log into [Netlify](https://app.netlify.com)
3. Click **"Add new site"** → **"Import an existing project"**
4. Select this GitHub repo
5. Publish directory: `.` (root)
6. Click **Deploy**

No build step required — this is a static HTML/CSS/JS site.

## 📁 Project Structure

```
├── index.html          # Main page (single-page site)
├── styles.css          # Complete design system
├── script.js           # Animations, navigation, form handling
├── images/             # Site images
│   ├── hero-bg.png
│   ├── clean-kitchen.png
│   └── pressure-washing.png
├── robots.txt          # Search engine crawler rules
├── sitemap.xml         # XML sitemap for SEO
├── netlify.toml        # Netlify deployment config
└── README.md
```

- Fully responsive (mobile, tablet, desktop)
- SEO optimized (JSON-LD structured data, meta tags, sitemap)
